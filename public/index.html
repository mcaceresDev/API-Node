Acerca de

Este es un proyecto de api sencilla desarrollado en Node js y conectado a una base de datos MySQL. Este demuestra una estructura básica de proyecto (scaffold), con archivos separados segun su función para un CRUD de artículos. El código usa javascript como lenguaje base y el paquete mysql para las consultas a la base de datos.

Tecnologías

  • Javascript
  • Express (Framwork de node)
  • MySql
  • Dotenv (para configurar variables de entorno)
  • Cors (Para habilitar cross-origin resource sharing)

Endpoints

Artículos

  • GET /items/all Obtiene todos registros
  • GET /items/1 Obtiene un regitro segun el id pasado por url
  • POST /items/new Guarda un nuevo registro en la BD (Debes enviar los datos "nombre, precio y stock" desde un formulario o desde un software de test como postman)
  • PUT /items/put/1 Actualiza un registro segun el id enviado por url (Debes enviar los datos "nombre, precio y stock" la nueva info desde un formulario o desde un software de test como postman)
  • DELETE /items/del/1 Elimina un registro segun el id pasado por url
